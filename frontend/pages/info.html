Projektin kuvaus

Tämä projekti toteuttaa selainpohjaisen dashboardin, joka vastaanottaa Raspberry Pi:n lukemia anturitietoja ja esittää ne käyttäjälle reaaliaikaisesti.

Tämän lisäksi käyttäjän tulee määrittää haluamansa kaupunki, jonka tämänhetkinen säätila haetaan OpenWeatherin palvelusta ja esitetään käyttäjälle.

Frontend on rakennettu HTML, CSS ja JavaScript -tekniikoilla, ja datan lähde on omassa verkossa oleva laitteisto (Arduino / Raspberry Pi).

Reaaliaikainen ulkolämpätila haetaan OpenWeatherMap API 2.5:sta ja tallennetaan localStorageen.
